# Regex Tutorial

A tutorial on explaining regular expressions. As an example, sses the expression
```
const regex = /^([a-z0-9_\.-]+)@([\da-z\.-]+)\.([a-z\.]{2,6})$/
``` 
for validating an email and goes through each component of the regex step by step.
